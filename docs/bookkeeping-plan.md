# BookEase – प्राइवेसी-फर्स्ट बुककीपिंग ऐप

## टेक्नॉलजी स्टैक
- React Native + Expo + TypeScript  
- डेटाबेस: Realm (लोकल, एन्क्रिप्टेड, ML रेडी)  
- लॉगिन: बायोमेट्रिक्स या 6-अंकीय पासकोड (Expo SecureStore)  
- सिंक: पीअर-टू-पीअर → QR स्कैन या ब्लूटूथ → एन्क्रिप्टेड JSON → ऑटो मर्ज (लेटेस्ट विन्स)  
- बैकअप & रिस्टोर: Realm → एन्क्रिप्टेड .zip → Google Drive (ऑटो अपलोड/डाउनलोड)

## मुख्य फीचर्स
1. डैशबोर्ड – कुल बैलेंस, आज का फ्लो  
2. ट्रांजेक्शन लिस्ट – ऐड/एडिट/डिलीट (हरा-लाल आइकन)  
3. रिपोर्ट – P&L चार्ट (Recharts/Victory), PDF एक्सपोर्ट  
4. ऑडिट लॉग – हर बदलाव का रिकॉर्ड  
5. डार्क/लाइट मोड + एक्सेसिबिलिटी  

## शुरू करने की कमांड
```
	npx create-react-native-app BookEase --template typescript
	cd BookEase
	expo start
```
फोन में Expo Go ऐप → QR स्कैन → लाइव देखो

## गोल
100 % प्राइवेसी, क्लाउड नहीं, फिर भी हर डिवाइस पर डेटा।
